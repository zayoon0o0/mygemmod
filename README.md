# About 
This Mod adds Red Gems to Minecraft, it also has red gem tools and armor! 
the ore spawns rarely in the End, making it an end ore. it also includes Amethyst Armor and Tools!

# About the Red Gem
The Red Gem is a super rare ore found in the end islands, it has properties better then netherite and a very low cooldown.

# About the Amethyst Addtions
The Amethyst tools and armor are super OP with no cooldown for the tools and also have some magical powers (WORK-IN-PROGRESS), the downside of them is that they have **VERY** low durability.
